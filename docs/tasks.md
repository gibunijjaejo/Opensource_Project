## ✅ 시간표 이미지 업로드 (OCR)

**완료 조건**

- jpg/jpeg/png 업로드 → Mistral Pixtral OCR → 과목 자동 매칭
- 매칭된 과목이 수강이력에 자동 저장
- 업로드 즉시 202 반환 (비동기 처리)

**의존성**

- 회원 인증 (JWT 필요)
- OCR 서비스 컨테이너 실행 중

## ✅ 배포 환경 구성

**완료 조건**

- `docker-compose.yml` — 운영용, 헬스체크 + 의존성 조건 설정
- `docker-compose.dev.yml` — 개발용, `--reload` + HMR + 코드 마운트
- `Makefile` — `make dev / prod / down / logs / ps` 단축 명령어
- `.env.example` — 환경변수 템플릿 제공
- `SECRET_KEY`, `ADMIN_SECRET_KEY` 하드코딩 제거 → `.env`로 분리
- `pre-deploy.sh` — 필수 환경변수 배포 전 검증

- 없음 (인프라 태스크)
